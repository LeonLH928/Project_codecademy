This project is to show implementations of encapsulation's concepts in Object-oriented programming (This is implemented in C++) 
As project's name suggests, this simulates basic functionalities of a bank account, including depositing, withdrawing, and transferring.
This project uses knowledge of encapsulation like access specifiers, nested class, etc.
For information about what this project can do, please check the Bank.hpp in Bank and BankAccount.hpp in BankAccount
