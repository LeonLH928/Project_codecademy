This is a Codecademy project named Skyroute

Here are some brief descriptions:
- This project helps to find the shortest route from one landmark to another landmark.
- This project implements breadth-first search, depth-first search, and Python dictionaries.
- graph_search.py has the bfs() and dfs() functions implemented in Python.
- vc_metro.py contains a graph of all stations in the Vancouver metro system.
- vc_landmarks.py contains a dictionary of Vancouver landmarks mapped to their nearest metro station(s).
- landmark_choices.py contains a dictionary of letters of the alphabet mapped to landmarks to make it easier for users to make a selection.

Here are functionalities of Skyroute in skyroute.py:
- skyroute is the function containing the full program. 
- This program can take a starting point, an ending point, updating any closed stations from the user's input
